# ⚡ React Boilerplate

Boilerplate project untuk memulai pengembangan aplikasi React dengan stack modern:

- **React Router** → Routing antar halaman
- **Tailwind CSS** → Styling cepat berbasis utility
- **Axios** → HTTP client untuk API
- **Day.js** → Manipulasi tanggal/waktu
- **Framer Motion** → Animasi interaktif

## 🚀 Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/Bangdams/react-boilerplate.git
cd react-boilerplate
```
